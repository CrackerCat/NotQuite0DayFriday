  • Only already-installed binaries will be updated. The patch will automatically discover and upgrade any components that are installed on your system, but will not add any components or change your configuration.

  • You cannot use this patch to install the software. You can only use it to upgrade an existing installation.

  • You cannot use this patch to upgrade from an evaluation version to the registered version. Use the evaluation patch to upgrade your evaluation version.
